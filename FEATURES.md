# CRM Break Wave - Новые функции

## 1. Отслеживание оставшихся занятий (Lesson Counter)

### Модель Student
- `lessons_count` - общее количество занятий в абонементе
- `lessons_remaining` - оставшиеся занятия

### Автоматический учет
- При создании студента `lessons_remaining = lessons_count`
- При отметке "present" - списывается 1 занятие
- При изменении статуса с "present" на другой - занятие возвращается
- При оплате - добавляются занятия к `lessons_remaining`

### UI
- Индикатор оставшихся занятий в списке учеников
- Прогресс-бар в карточке ученика
- Предупреждения при ≤2 занятиях

## 2. Внеплановые посещения (Extra Attendance)

### API
- `POST /api/extra-attendance` - отметить внеплановое посещение
  - Параметры: student_id, date, time, status, notes, deduct_lesson
  - Можно выбрать, списывать ли занятие с абонемента

### Модель Attendance
- `is_extra` - флаг внепланового посещения
- `attendance_date` - фактическая дата посещения
- `attendance_time` - фактическое время посещения
- `lesson_id` теперь nullable (для внеплановых)

### UI
- Кнопка "⭐ Внеплановое" в карточке ученика
- Модальное окно с выбором даты/времени/статуса
- Галочка "Списать занятие"

## 3. История посещений

### API
- `POST /api/students/{id}/attendance-history`
  - Возвращает все посещения (включая внеплановые)
  - Статистика: total_scheduled, extra_lessons, attendance_rate

### UI
- Кнопка "📋 История" в карточке ученика
- Модальное окно со списком всех посещений
- Бейдж "⭐ Внеплановое" для extra посещений
- Статистика в заголовке

## 4. Ежедневные уведомления

### Модель DailyNotificationLog
- Отслеживает отправленные уведомления (раз в день)
- Поля: coach_id, notification_type, date

### Планировщики (main.py)
- `daily_notification_scheduler()` - отправка сводки в 9:00
- `lesson_reminder_scheduler()` - напоминания о неотмеченных занятиях каждые 5 мин

### API
- `POST /api/coach/daily-summary` - получить сводку для тренера

### Бот
- Команда `/summary` - запросить сводку вручную
- Автоматическая отправка каждый день в 9:00
- Содержит:
  - Просроченные абонементы
  - Заканчивающиеся абонементы (≤3 дней)
  - Ученики без занятий
  - Ученики с 1-2 занятиями
  - Расписание на сегодня

## 5. Управление группами

### API
- `POST /api/current-lesson/add-student` - добавить ученика к текущей/следующей группе
- `POST /api/groups` - получить все группы по времени

### UI (в планах)
- Добавление ученика к группе на экране быстрого занятия

## 6. Обновления интерфейса

### Dashboard
- Предупреждения о заканчивающихся занятиях
- Интеграция с daily summary

### Список учеников
- Индикатор оставшихся занятий
- Бейдж "⚠️ N занятий" при ≤2
- Бейдж "❌ Нет занятий" при 0

### Карточка ученика
- Прогресс-бар использования абонемента
- Предупреждения (alert-warning/alert-danger)
- Статистика посещаемости
- 4 кнопки действий:
  - ✏️ Редактировать
  - 💰 Оплата
  - ⭐ Внеплановое
  - 📋 История
