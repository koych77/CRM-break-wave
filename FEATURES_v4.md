# CRM Break Wave - Версия 4.0

## 1. Разное время занятий по дням недели

### Модель
- `lesson_times` (JSON): `{"1": "18:00", "3": "19:30"}`
- Каждый день может иметь свое время
- Метод `get_lesson_time_for_day(day_of_week)`

### UI
- При выборе дней недели автоматически генерируются поля времени для каждого дня
- Можно установить разное время для разных дней

## 2. Система залов (Locations)

### Модель Location
- `id`, `coach_id`, `name`, `address`, `is_active`
- Связь с Student (primary location)
- Связь с Lesson (where lesson took place)
- Связь с Attendance (where student attended)

### API
- `GET /api/locations` - список залов
- `POST /api/locations/create` - создать зал
- `POST /api/locations/{id}/update` - обновить
- `POST /api/locations/{id}/delete` - удалить

### UI
- Выбор зала в форме ученика (select)
- При отметке занятия можно выбрать зал

## 3. Подробная статистика

### API
- `POST /api/statistics` - детальная статистика
  - Параметры: period (week/month/year)
  - Фильтр по залу

### Данные статистики
- Общая сводка: ученики, занятия, посещаемость
- По дням недели: график посещаемости
- По залам: сравнение залов
- Возрастные группы: дети, подростки, взрослые
- Динамика: последние 6 месяцев

### UI
- Новый экран "📊 Статистика"
- Табы: Месяц/Неделя/Год
- Графики и диаграммы

## 4. Поиск

### API
- `POST /api/search`
  - Параметр: query (минимум 2 символа)
  - Ищет по: имени, никнейму, телефону, телефону родителя

### UI
- Новый экран "🔍 Поиск"
- Быстрый поиск с debounce (300ms)
- Результаты с основной информацией
- Переход к карточке ученика

## 5. Исправление валюты
- Везде заменено ₽ на Br (белорусские рубли)
- В формах, списках, статистике

## Миграция базы данных

### Новые таблицы
```sql
CREATE TABLE locations (
    id INTEGER PRIMARY KEY,
    coach_id INTEGER NOT NULL,
    name VARCHAR(200) NOT NULL,
    address VARCHAR(500),
    is_active BOOLEAN DEFAULT 1,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

## Использование

### Создание зала
1. Залы создаются через API (позже можно добавить UI)
2. При создании ученика выбирается зал из списка

### Настройка разного времени
1. Открыть форму ученика
2. Выбрать дни недели
3. Для каждого дня появится поле времени
4. Можно установить разное время для разных дней

### Просмотр статистики
1. На главной нажать "📊 Статистика"
2. Выбрать период (Месяц/Неделя/Год)
3. Просмотреть графики

### Поиск ученика
1. На главной нажать "🔍 Поиск"
2. Ввести имя, телефон или никнейм
3. Выбрать из результатов
